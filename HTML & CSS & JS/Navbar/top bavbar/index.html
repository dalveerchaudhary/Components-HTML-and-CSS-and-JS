Nav Bar Reveal On Scroll

Simple JS and HTML/CSS for a Navigation bar which reveals itself when the user scrolls a certain amount of pixels down the screen. For example, this is useful for when you wish to only introduce a Nav bar when the user scrolls past a hero header etc. This can also be adapted for mobile by just changing the width values for certain breakpoints.

If you keep scrolling down, you can add more content and the nav bar will still appear on top in a fixed position


Change the 'y>=200' value in the JS file to trigger at which point you want to reveal the navigation bar.
